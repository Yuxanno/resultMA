# Скрипт создания учеников 7-го класса

## Описание
Этот скрипт автоматически создает 30 учеников 7-го класса группы А и добавляет их во все группы по всем предметам.

## Что делает скрипт:
1. ✅ Создает 30 учеников с узбекскими именами
2. ✅ Присваивает им 7-й класс
3. ✅ Генерирует случайные телефонные номера
4. ✅ Создает уникальные токены профиля для каждого ученика
5. ✅ Создает группы "7-A" для всех активных предметов (если не существуют)
6. ✅ Добавляет всех учеников во все группы по всем предметам

## Запуск скрипта

### Из папки server:
```bash
cd server
npm run create-7th-grade
```

### Или напрямую:
```bash
cd server
npx tsx src/scripts/create7thGradeStudents.ts
```

## Требования
- MongoDB должна быть запущена
- Должен существовать хотя бы один активный филиал
- Должны быть созданы предметы (можно создать через `npm run seed`)

## Результат
После успешного выполнения вы увидите:
- Список созданных учеников с их именами и телефонами
- Количество созданных групп
- Количество связей ученик-группа
- Итоговую статистику

## Примечания
- Скрипт безопасен для повторного запуска - не создаст дубликаты групп
- Все ученики получают доступ ко всем активным предметам
- Имена генерируются случайно из списка узбекских имен
- Телефонные номера генерируются в формате +998XXXXXXXXX
