# Исправление проблемы с отображением учителей у филиал админа

## Проблема
Филиал админ не видит учителей, созданных из админ панели.

## Причина
У филиал админа не установлен `branchId`, поэтому при запросе учителей фильтр не работает корректно.

## Решение
Скрипт `fixFilAdminBranch.ts` автоматически:
1. ✅ Находит всех филиал админов без `branchId`
2. ✅ Назначает им первый доступный филиал
3. ✅ Проверяет всех учителей без `branchId`
4. ✅ Назначает им филиал

## Запуск

### Вариант 1: Через npm
```bash
cd server
npm run fix-filadmin-branch
```

### Вариант 2: Через bat файл
```bash
fix-filadmin.bat
```

## После запуска
1. Перезагрузите страницу в браузере
2. Учителя должны отображаться у филиал админа

## Примечание
- Скрипт безопасен для повторного запуска
- Не изменяет данные, если они уже корректны
- Показывает подробный лог всех изменений
